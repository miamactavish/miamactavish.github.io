Shader Educational Tool

This is a tool forked from BKcore's Shdr project, which is an online tool that allows viewing and editing of OpenGL shaders via WebGL. This project was adapted for an advanced computer science course delivered at the high-school level, in its Computer Graphics unit. I made various modifications to the original project, including:


  • Ability to upload, and modify shaders for, 3D models in .obj format. Supports switching between different 3D models at runtime
  • Ability to apply postprocessing effects
